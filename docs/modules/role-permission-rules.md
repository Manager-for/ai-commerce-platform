# Role & Permission Rules

## System Roles

### Owner

Access:
- Full Access

Can:
- Manage Subscription
- Manage Users
- Manage Roles
- View Analytics
- Approve Orders
- Manage Integrations

---

### Admin

Access:
- Operational Access

Can:
- Manage Users
- Manage Products
- Manage Customers
- Manage Orders

Cannot:
- Change Subscription
- Delete Tenant

---

### Manager

Can:
- View Dashboard
- Approve Orders
- View Reports

Cannot:
- Manage Users
- Manage Subscription

---

### Sales Executive

Can:
- Create Customer
- Create Order
- View Assigned Customers

Cannot:
- View Financial Reports
- Manage Users

---

### Support Agent

Can:
- Reply Customers
- View Orders

Cannot:
- Approve Orders
- Manage Products

---

### AI Agent

Can:
- Generate Replies
- Create Draft Orders
- Generate Summaries

Cannot:
- Approve Orders
- Delete Data
- Change Settings
